📱 HOW TO RUN

🛠️ DATA PERSISTENCE
--------------------
- Uses browser LocalStorage
- Data persists across page refresh
- Sample data pre-loaded
- Reset button available (bottom-right)

🎮 HOW TO USE
-------------
AS A GUEST:
1. Browse rooms from Home or Rooms page
2. Click "Book Now" on any room
3. Fill booking form (dates, guests)
4. Proceed to payment simulation
5. View booking confirmation

AS ADMIN:
1. Click "Admin" in navigation
2. Login with admin/admin123
3. View dashboard statistics
4. Manage bookings (confirm/cancel)
5. Add/edit rooms
6. View reports

📊 DATA STORAGE
---------------
The system stores:
- Bookings (confirmed/pending/cancelled)
- Room inventory
- User accounts
- Admin settings

All data saved in browser's LocalStorage.

🔧 CUSTOMIZATION
----------------
1. Change resort name in index.html (line 13)
2. Update colors in style.css (:root variables)
3. Modify room prices in script.js (roomTypes object)
4. Add your team details in footer (index.html)
